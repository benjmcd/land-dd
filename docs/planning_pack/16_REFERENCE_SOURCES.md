# 16 Reference Sources and Standards

This file is a seed list for source-registry and architecture validation. Each source still requires a production license/terms review.

## Storage and geospatial standards

| Reference | URL | Use |
|---|---|---|
| PostgreSQL | https://www.postgresql.org/ | system-of-record database |
| PostGIS | https://postgis.net/ | spatial types, indexes, and geospatial operations in Postgres |
| STAC | https://stacspec.org/ | catalog model for spatiotemporal assets |
| GeoParquet | https://geoparquet.org/releases/v1.1.0/ | exchange/archive format for geospatial vector data |
| OGC Cloud Optimized GeoTIFF | https://www.ogc.org/standards/ogc-cloud-optimized-geotiff/ | cloud-friendly raster format |

| Reference | URL | Use |
|---|---|---|
| USGS The National Map | https://www.usgs.gov/programs/national-geospatial-program/national-map | elevation, hydrography, boundaries, land cover, transportation, base layers |

| USFWS National Wetlands Inventory | https://www.fws.gov/program/national-wetlands-inventory | wetland/deepwater screening |

| USGS MRDS | https://mrdata.usgs.gov/mrds/ | historical mineral occurrence screening |
