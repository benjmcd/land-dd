This review records the rights decisions required before Bologna candidate sources can
be promoted. It is not a license review and does not approve cache, redistribution,
export, AI use, raw-data handling, fixture capture, runtime use, or report use.

## Required Decisions

Every promoted source must answer:

- terms reference and effective date;
- source version or publication date;
- update cadence and freshness class;
- license, commercial-use, redistribution, cache, export, raw-data, AI-use, and
  attribution decisions;
- retrieval metadata policy;
- source-failure, no-data, caveat, CRS/precision, field allowlist, and field denylist
  policies;
- fixture capture and report-use policy.
