## Why NOT_EVALUATED

- Assessor data varies significantly by county in schema, access method, update cadence, and license.
- Owner name, owner address, and assessed value are fields with privacy implications in some jurisdictions; a per-county field policy decision is required before integration.
- The private MVP parcel screening goal is met by DS-010 (geometry + PIN + ZONING) without assessor data for the initial build.
- Official county web sources exist for all three private-MVP counties, but each still needs endpoint-level access review, field-policy approval, and connector design before DS-011 can be promoted.
