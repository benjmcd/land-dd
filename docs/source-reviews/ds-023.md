## Scope and status

DS-023 covers local zoning ordinance PDFs: allowed uses, minimum lot sizes, setbacks, overlays, and other text-derived zoning attributes. Each county or municipality publishes ordinances at different URLs, in different formats, and with different amendment schedules. For the private MVP:

- **Chatham County recorded-fixture connector implemented 2026-06-10.** `ChathamZoningRecordedConnector` uses a human-reviewed snapshot of the Chatham County UDO (effective 2025-07-01) to classify zoning district codes from parcel/CAMA data. No live HTTP requests; no raw PDF redistribution; no owner or value fields exposed.
- **Scope: Chatham County UDO.** Covers 13 district codes (RA, RR, R-1, R-2, R-3, MHP, B-1, B-2, I, I-1, I-2, PD, UNZONED) with residential-use screening and fail-closed unknown/needs-review evidence for unrecognized codes.
- **Brunswick County recorded-fixture connector implemented 2026-06-11.** `BrunswickZoningRecordedConnector` uses a human-reviewed snapshot of the Brunswick County UDO (most recent revision 2024-08-19) to classify zoning district codes. Covers 12 base districts (RR, R-7500, R-6000, SBR-6000, MR-3200, C-LD, N-C, C-I, RU-I, I-G, MI, CP) and 5 overlay districts (CZ, ED, PD, TO, WQP). No live HTTP requests; no raw PDF redistribution; no owner or value fields exposed.
- **Live PDF connector remains deferred.** The Chatham County website returns HTTP 403 on programmatic fetch; live ingestion remains blocked pending per-county document-reuse review.
- **Status: approved-with-restrictions.** This approval covers the recorded-fixture approach for Chatham and Brunswick County UDOs; it does not cover raw PDF redistribution, full-text caching, live connector ingestion, or Buncombe County zoning.

## Why fixture-backed

- Zoning ordinance PDFs are public records but parsing them reliably requires OCR and structure-extraction work that is deferred beyond the private MVP.
- License terms vary: some jurisdictions explicitly permit reuse; others have disclaimers that require per-county evaluation.
- The fixture approach proves the evidence/claim pipeline end-to-end without resolving per-county license and parsing questions.
- Official county zoning/ordinance sources exist for all three private-MVP counties, but live production use still needs document reuse review, parsing scope, amendment tracking, and jurisdiction handling before DS-023 can be promoted.

## Production decision

- Approved for fixture-only development? yes — StaticZoningFixtureConnector active in private MVP

- Approved for recorded-fixture connector use (Brunswick County UDO)? yes — `BrunswickZoningRecordedConnector` implements the recorded-fixture path (same policy basis as Chatham); screening-only; redistribution restricted; connector surfaces: `immediate_operator_api`, `request_time_orchestration`; implemented 2026-06-11
