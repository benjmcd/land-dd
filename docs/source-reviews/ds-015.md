# Data Source Review: DS-015 NC Geological Survey 1985 Geologic Map

## Source identity

- Source registry ID: DS-015
- Source name: State geological survey
- Organization: North Carolina Geological Survey, North Carolina Department of Environmental Quality
- Reviewed scope: 1985 North Carolina Geologic Map FeatureServer, Map Units layer only
- Service URL: https://services2.arcgis.com/kCu40SDxsCGcuUWO/arcgis/rest/services/Geologic_Map_of_North_Carolina/FeatureServer
- Query layer: https://services2.arcgis.com/kCu40SDxsCGcuUWO/arcgis/rest/services/Geologic_Map_of_North_Carolina/FeatureServer/3
- Date reviewed: 2026-06-11
- Review status: approved-with-restrictions

## Official authority

- The NC DEQ interactive geologic content page lists the 1985 North Carolina Geologic Map as an interactive digital equivalent of the official 1985 State Geology Map.
- The ArcGIS service description says the feature layer represents the digital equivalent of the official State Geology Map of 1985.
- The service description attributes the GIS dataset to NCGS within NC DEQ, in cooperation with the NC Center for Geographic Information and Analysis.
- The service states the data represents the official statewide 1:500,000 geologic map, digitized from 1:250,000 base maps.
- The service states this geology is deprecated and that no current replacement covers the whole state.
- The Map Units layer is a polygon Feature Layer, supports JSON queries, and exposes UnitLabel as its display field.

## Scope decision

Approved scope is geologic map-unit context only.

The connector may query bounded EPSG:4326 bbox envelopes against the Map Units layer and store:

- Object IDs
- Unit labels
- Belt, type, formation, and description text
- Result count
- Query bbox
- Map year and deprecated-data flag
- Retrieval metadata, source URL, caveat, and source failures

The connector must not use this source to infer:

- Landslide, sinkhole, radon, or other geologic hazard determinations
- Mineral resources, mineral rights, mine hazards, or extraction value
- Foundation, engineering, geotechnical, or site suitability
- Buildability, permitting outcome, appraisal, lending, insurance, or investment suitability

Absence of a returned map unit or absence of a particular unit label must not be treated as proof that no geologic constraint exists.

## Explicit non-selection

The NC landslide inventory was reviewed but not selected for this DS-015 readiness slice.

Current live probes on 2026-06-11 showed:

- `https://maps.deq.nc.gov/arcgis/rest/services/DEMLR/North_Carolina_Landslide_Inventory/FeatureServer/0?f=json` returned HTTP 200 with an ArcGIS error payload: `code=500`.
- A simple layer query with `where=1=1`, `outFields=OBJECTID,SM_Name`, `returnGeometry=false`, and `f=json` returned HTTP 200 with an ArcGIS error payload: `code=500`, message `Error performing query operation`.

Because the landslide source did not provide a stable machine-readable JSON/query surface in this pass, it remains out of connector scope. Any future landslide/hazard connector needs its own source review, tests, caveats, and source-failure behavior.

## Product caveats

- The source is historical, statewide, generalized geologic-map data.
- It is deprecated and not parcel-scale geology.
- It was digitized from smaller-scale base maps and is not a substitute for site-specific geologic, geotechnical, engineering, hazard, or professional review.
- It does not determine mineral ownership, resource value, buildability, suitability, insurability, lending suitability, or investment suitability.
