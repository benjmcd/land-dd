# Data Source Review: DS-008 USGS MRDS

## Source identity

- Source registry ID: DS-008
- Source name: USGS MRDS
- Organization: U.S. Geological Survey
- Reviewed scope: Mineral Resources Data System historical mineral-occurrence records
- Homepage: https://mrdata.usgs.gov/mrds/
- WFS endpoint: https://mrdata.usgs.gov/services/wfs/mrds
- Date reviewed: 2026-06-11
- Review status: approved-with-restrictions

## Official authority

- The USGS MRDS page identifies MRDS as a large, formerly active collection of mineral-resource records and states that systematic updates ended in 2011.
- The same official page provides web services for MRDS, including WFS, WMS, and WCS, plus downloadable shapefile, CSV, and KML artifacts.
- The MRDS WFS `GetCapabilities` endpoint exposes feature type `mrds` in EPSG:4326.
- The MRDS WFS `DescribeFeatureType` endpoint exposes compact fields: `dep_id`, `site_name`, `dev_stat`, `fips_code`, `huc_code`, `quad_code`, `url`, and `code_list`.
- USGS copyright/credits policy treats most USGS-authored information as public domain while still requiring source credit and noting that third-party materials may carry separate restrictions.

## Approved connector scope

The approved connector may:

- Query MRDS WFS `mrds` records intersecting a bounded EPSG:4326 bbox.
- Use WFS 1.0.0 lon/lat bbox order to match existing connector bbox contracts.
- Store `dep_id`, site name, development status, commodity code list, record URL, count, bbox, retrieval timestamp, source URL, and caveat.
- Emit explicit source-failure evidence for network errors, WFS exceptions, malformed XML, or possible truncation when the response reaches `max_features`.

The connector must not:

- Infer mineral rights, private mineral severance, mine hazards, resource value, extraction feasibility, environmental liability, appraisal value, lending suitability, investment suitability, or buildability.
- Treat absence of MRDS records as absence of mineral rights or geologic/mineral risk.
- Bulk download or redistribute MRDS CSV/shapefile/KML artifacts in the live connector path.

## Product caveats

- MRDS is historical screening data; systematic updates ceased in 2011.
- MRDS records vary in accuracy and completeness.
- A nearby MRDS occurrence is not parcel-specific proof of mineral ownership, mine hazards, contamination, or economically viable resources.
- Absence of records in a bbox does not rule out private mineral severance, unrecorded hazards, or local geologic constraints.
